# Расписание МАУ (API)

### Сборка

Создать .env файл в корне

```sh
HOST=hostname
USER=username
PASSWORD=password
DATABASE=database
```

Установить зависимости

```sh
npm install
```

Запуск проекта

```sh
npm run dev # сборка для разработки с горячим обновлением
npm run start # старт api
```

Документация 

```sh
http://localhost:8080/api
```

## Основные используемые технологии и библиотеки

-   [express](https://github.com/expressjs/express)
-   [mysql2](https://github.com/sidorares/node-mysql2)
